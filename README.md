# fhir.ts

## Open source typescript library providing FHIR utilities

# Install

```
npm install @arkhn/fhir
yarn add @arkhn/fhir
```

# Usage

```typescript
import { Node } from '@arkhn/fhir'

const node = new Node()
```

## Build and publish

```sh
yarn build # builds the library
yarn publish # publish the library on npm
```
